# OpenAegis Training Manual (Basa Jawa)

This edition is prepared for Javanese readers.

## Training Sequence
1. Login demo users.
2. Run simulation workflow.
3. Run live workflow.
4. Approve pending request.
5. Verify audit evidence.

## Skills to Demonstrate
- Explain why a workflow was blocked.
- Find approval and approver details.
- Find model route and risk controls.
- Read evidence IDs in audit logs.

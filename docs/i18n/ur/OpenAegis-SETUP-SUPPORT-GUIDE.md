# OpenAegis Setup & Support Guide (????)

This edition is prepared for Urdu readers.

## Setup
1. `npm install`
2. `npm run typecheck`
3. `npm run test`
4. `npm run build`
5. `npm run smoke:pilot`

## Pilot Demo
- Run `node tools/scripts/pilot-demo.mjs`
- Open UI at `http://127.0.0.1:4273`

## Support
When reporting issues include:
- command executed
- full error output
- expected vs actual result
- screenshot/log reference
